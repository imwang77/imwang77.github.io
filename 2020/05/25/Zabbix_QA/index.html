Zabbix_QA

zabbix初始登陆密码是什么？

用户名：Admin
密码：zabbix

cannot set resource limit: [13] Permission denied

关闭selinux即可或者更改selinux的策略

selinux 0
systemctl disable selinux

服务器已经监听了80端口但是telnet不通？

Centos7 关闭firewalld

systemctl stop firewalld
systemctl disable firewalld

Centos6 关闭iptables

service iptables stop
chkconfig off iptables

zabbix agent的安装方式？

官网地址:https://www.zabbix.com/cn/download?zabbix=3.0&os_distribution=centos&os_version=6&db=&ws=

  • 从官网下载安装包解压后即可使用

下载路径：https://www.zabbix.com/cn/download_agents


根据系统和ZabbixServer版本不同选择下载
上传到服务器上，解压即可使用

  • 安装官网yum源，通过yum安装

先选择服务器和ZabbixServer的版本，根据版本生成对应的repo库

根据系统版本的不同，启动命令和开机自启服务命令也会不同
