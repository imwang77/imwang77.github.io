Windows server服务器如何允许同一用过同时拥有多个远程连接会话？

在多人登录的情况下，默认第二个人或把第一个人踢掉，为了便于使用，将系统设置成可以多人同时同账户进行登录，如下：
打开cmd，输入gpedit.msc\ 组策略，找到【计算机设置】–【管理模板】–【windows组件】–【远程桌面服务】–【远程桌面会话主机】–【连接】

Windows 路由表中 在链路上 是什么意思？

  • 在链路上是指通过2层链路可以直接通达的路由。
  • 有接口地址的是指通过3层路由转发的路由。
  • 他们分别对应的路由一个是直达路由一个是静态或动态路由，直达路由比动态或静态路由的优先级都要高。
  • 同一优先级别的路由 选择跳数（metric）小的路由。

Win Server可以ping通别人，但别人不能ping通自己？

  • 服务器在网络适配器的ipv4网关 无法设置，配置了点击确定后重新打开还是空白
  • 但在cmd中ipconfig可以看到网关
  • 此服务器是双网卡
  • 重启服务器网卡后，问题解决
